4. "us2020_propuesta_estudiantes_ngram.ipynb": Archivo con un set de de datos compuesto por los candidatos Joe Biden, Dondald Trump y Mike Pence. Aplicación de todos los (5) pasos de pretratamiento mencionados en el informe. PCA con bigramas. Evaluación del modelo Multinomial Naive Bayes.

5. "us2020_propuesta_estudiantes_LogisticRegression.ipynb": Archivo con un set de de datos compuesto por los candidatos Joe Biden, Dondald Trump y Mike Pence. 6. "us2020_propuesta_estudiantes_LinearSVM.ipynb": Archivo con un set de de datos compuesto por los candidatos Joe Biden, Dondald Trump y Mike Pence. Aplicación de todos los (5) pasos de pretratamiento mencionados en el informe. PCA con unigramas. Evaluación del modelo Linear SVC.

7. "us2020_propuesta_estudiantes_BarakObama.ipynb": Archivo con un set de de datos compuesto por los candidatos Joe Biden, Dondald Trump y Barack Obama. 8. "us2020_propuesta_estudiantes_BarakObama_LR.ipynb": Archivo con un set de de datos compuesto por los candidatos Joe Biden, Dondald Trump y Barack Obama. Aplicación de todos los (5) pasos de pretratamiento mencionados en el informe. PCA con unigramas. Evaluación del modelo Logistic Regression.
